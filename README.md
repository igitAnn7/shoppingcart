# Interactive Shopping Cart 🛒

A browser based shopping cart application built with vanilla 
JavaScript, HTML and CSS.

## Live Demo
(https://igitAnn7.github.io/shopping-cart)

## Features
- Browse and add items to cart with custom quantities
- Real time cart updates as items are added or removed
- Dedicated checkout page showing full order summary
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## What I Learned
- Dynamic DOM manipulation with JavaScript
- Managing state across multiple pages
- Handling user interactions and real time UI updates
- Building a practical real world application feature
